## psh_download_execute_psh

This is a "Downloader" payload type, meaning it will fetch the payload from a url.
It is NOT an agent

### Downloader
  
This downloader will download a powershell script and executes it with IEX

Parameters:
- ps1_url: URL where the Powershell payload is located
