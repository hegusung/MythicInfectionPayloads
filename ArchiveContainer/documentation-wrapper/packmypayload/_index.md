## PayMyPayload

PackMyPayload is a well-known github project allowing to create archives of various types for payload delivery

### Wrapper
  
Currently, this wrapper allows to create archives of the following types:
- zip
- 7zip
- iso
- cab

The payload included in the archive can have its name defined by the "payload_name" parameter.

Additional files can be added
